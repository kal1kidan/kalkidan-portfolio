## 📖 Overview
A fully animated, premium React portfolio website designed for a modern digital presence. This portfolio elegantly showcases my professional identity as a **Host, Digital Marketer, Content Creator, and Software Engineer**. 

Built with a warm "Luxury Brown" aesthetic, the site features:
- **Cinematic Animations:** Powered by Framer Motion for smooth, editorial-style transitions and scroll reveals.
- **Vertical Video Showcase:** A custom-built, TikTok-style portfolio section optimized for native 9:16 vertical video playback.
- **Responsive Design:** Mobile-first architecture ensuring a seamless, premium experience across all devices.
- **Modern Tech Stack:** React, Vite, and Vanilla CSS for blazing fast performance and clean code.

## 🛠️ Technologies Used
- **Frontend Framework:** React.js
- **Build Tool:** Vite
- **Animations:** Framer Motion
- **Styling:** Vanilla CSS (CSS Variables, Flexbox, CSS Grid)
- **Icons:** Lucide React
- **Deployment:** Netlify

## 🚀 Running Locally
To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/kal1kidan/kalkidan-portfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd kalkidan-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
